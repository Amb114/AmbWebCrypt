### ✨ 特性

1.  **极强加密级别**：AES256_GCM 和 P-384 混合。跑的是硬件加速，100MB文件2.4秒搞定。
2.  **大文件不虚**：手机上能加密 **1.66GB**（1.55GiB）文件而**不用下APP**！网页直接吃。
3.  **无限克隆程序本体**：点那个“保存本页”，程序你就带走了，匿名去中心化快速传播。
4.  **智能还原**：忘记加密格式没关系，密码输对能自己给你试出来。忘记某文件后缀也能试出来。
5.  **首创汉字256/16进制**：这是我写的“现代诗”。发不了文件的地方直接起飞，比Base64隐秘、具备迷惑性。
6.  **双重压缩**：自带 WebP 预处理和 Gzip 压缩，图片30-160KB画质还行，体积减到更小。
7.  **自动化下载+解密外站资源**：可以小白点击链接就能自动下载+解密文件。

**不喜勿喷。传播的时候，只需要留着“AmbWebCrypt”这个名字不改就行，其他的都共享了。**

### ⚠️ 关于 Bug 和浏览器

**全是浏览器的锅。** 浏览器版本一定要用最新的，但有的最新版也有 Bug，我们全在最新的里面选。

*   **Chrome**：下载第二个文件会直接点不动（典型 Bug）。
*   **手机自带浏览器**：基本跑不动 Ed25519 签名。
*   **Aw，Snap！**：优先考虑换个性能好的设备，其次才是清理内存。
*   **Via 浏览器**：支持全部功能，是**完全体** [强]。（还得看手机，有的手机不是它）

**总结：如果在一个浏览器卡住了，就 Chrome、Edge、Via 互换，选对了就能起飞。**

### 💡 发掘隐藏功能

1.  **汉字起飞**：每个汉字 1 字节。在只能发文本的地方实现数据的隐秘“起飞”，复制汉字发出去，对方就能还原回文件。
2.  **救砖后缀**：把一个忘记后缀的文件拖进解密页，密码留空，有概率后缀就回来了。
3.  **AI 才是说明书**：点“保存本页”，把下载的文件后缀改成 **.txt** 发给 AI。让 AI 读代码，它能教你所有进阶参数和隐藏玩法。
