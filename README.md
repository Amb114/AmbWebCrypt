# AmbWebCrypt
AmbWebCrypt - 前端AES加密解密工具，支持GCM/P384/Ed25519算法，适配大文件处理。2026新年快乐。

为了方便我的日常生活，花一周时间也是做出了一个网页加密程序。（）

特性：

1.极强加密级别。（AES256_GCM和P-384）

2.手机上能加密1.66GB（1.55GiB）文件而不用下APP。

3.能无限克隆程序本体，匿名去中心化快速传播。（点那个“保存本页”）

4.忘记加密格式，密码输对能自己给你试出来，还带文件后缀，忘记某文件后缀也能试出来

5.首创汉字256/16进制，发不了文件也能起飞，比Base64安全，新，便捷100倍

6.加密的同时还自带webp和gzip压缩（30-160KB图片画质还行）

7.能自动化下载外站资源，让小白点击链接就能自动下载+解密文件

传播的时候，只需要留着“AmbWebCrypt”这个名字不改就行。其他的共享了（）
