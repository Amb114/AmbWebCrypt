### ✨ 核心特性

1.  **极高安全强度**：支持 AES-256-GCM 工业级加密及 P-384 (ECDH) 非对称加密。内置 Ed25519 数字签名验证，确保数据来源真实可靠。
2.  **超大文件支持**：手机端即可加解密 1.66GB (1.55GiB) 级别的大文件而无需下载 APP，全本地化操作，保护隐私。
3.  **首创汉字编码**：独创汉字 256/16 进制映射，1 个汉字对应 1 字节。在发不了文件、只能发文本的地方（如评论区、私信）直接“起飞”，比 Base64 安全、便捷、隐秘 100 倍。
4.  **去中心化克隆**：点击“保存本页”即可下载完整的单文件程序。程序本体即可匿名去中心化快速传播，支持无限克隆。
5.  **智能还原识别**：内置 30+ 种常用文件头识别。只要密码正确，还原时工具会自动尝试找回原始文件的后缀名。
6.  **多媒体预处理**：加密图片时可选自动转码为 WebP 并启用 Gzip 压缩，在大幅减小体积的同时保留可用画质。
7.  **自动化分发**：支持通过 URL 参数实现自动化下载外站资源并解密，让接收者点击链接即可直接触发下载。

传播时只需保留“AmbWebCrypt”名称。欢迎自由克隆、去中心化扩散，共享自由。
